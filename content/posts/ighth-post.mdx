# آشنایی با زبان برنامه‌نویسی سالیدیتی و توسعه قراردادهای هوشمند

سالیدیتی (Solidity) زبان برنامه‌نویسی اصلی برای توسعه قراردادهای هوشمند روی بلاک‌چین اتریوم است. در این مقاله، مفاهیم پایه‌ای و نحوه شروع کار با این زبان را آموزش می‌دهیم.

## مفاهیم پایه‌ای قراردادهای هوشمند

قراردادهای هوشمند، برنامه‌هایی هستند که روی بلاک‌چین اجرا می‌شوند و قوانین یک توافق را به صورت کد پیاده‌سازی می‌کنند. این قراردادها خودکار، غیرمتمرکز و غیرقابل تغییر هستند.

## ساختار یک قرارداد ساده در سالیدیتی

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract SimpleStorage {
    uint256 private storedData;
    
    function set(uint256 x) public {
        storedData = x;
    }
    
    function get() public view returns (uint256) {
        return storedData;
    }
}
